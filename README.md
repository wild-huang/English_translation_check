# 这是一个用来检查英语翻译的东西

> powered by AI

> 使用Opencode&XiaomiMimo构建

>使用flutter构建

# features

1.使用时必须接入AI的api，否则就是个看着好看的空壳子

2.兼容OpenAI和Anthropic（Anthropic没测试过，不过有这个软件的需求的应该也没必要用Anthropic）

3.支持Material Design 3，但是切换主题色就是个摆设什么时候想起来了我会去拷打ai的

4.其实检查这部分反而没啥好说的，就是几个分区吧，但是比直接问ai要来的更明确
